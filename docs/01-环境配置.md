# ⚙️ 环境配置指南

详细的构建环境准备和配置说明。

## 📋 系统要求

### 最低要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Ubuntu 20.04+ 或 Debian 11+ |
| **CPU** | 双核 x86_64 |
| **内存** | 4GB RAM |
| **磁盘** | 20GB 可用空间 |
| **权限** | Root (sudo) |

### 推荐配置

| 项目 | 推荐 |
|------|------|
| **操作系统** | Ubuntu 22.04 LTS |
| **CPU** | 四核或更多 |
| **内存** | 8GB+ RAM |
| **磁盘** | 50GB+ SSD |
| **网络** | 稳定的互联网连接 |

```bash
    curl \
```

### 分类说明

#### 1. 交互界面工具
```bash
sudo apt-get install dialog
```

#### 2. 交叉编译工具链
```bash
sudo apt-get install gcc-aarch64-linux-gnu
```

#### 3. RootFS 构建工具
```bash
sudo apt-get install mmdebstrap debootstrap qemu-user-static
```

#### 4. 内核编译工具

#### 5. 辅助工具
```bash

```

## 🔧 环境验证

### 1. 验证交叉编译器

```bash
aarch64-linux-gnu-gcc --version
```

**预期输出：**
```
aarch64-linux-gnu-gcc (Ubuntu/Debian X.X.X-XXubuntu1) X.X.X
```

### 2. 验证 mmdebstrap

```bash
which mmdebstrap
```

**预期输出：**
```
/usr/bin/mmdebstrap
```

### 3. 验证 QEMU

```bash
qemu-aarch64-static --version
```

**预期输出：**
```
qemu-aarch64 version X.X.X
```

### 4. 验证磁盘空间

```bash
df -h | grep -E "/$|/home"
```

**确保至少有 20GB 可用空间。**

## 🌍 配置国内镜像源（可选但推荐）

### Ubuntu 镜像源

```bash
sudo tee /etc/apt/sources.list << 'EOF'
# 阿里云镜像（推荐）
deb http://mirrors.aliyun.com/ubuntu/ jammy main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ jammy-updates main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ jammy-security main restricted universe multiverse

# 清华镜像（备选）
# deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy main restricted universe multiverse
# deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-updates main restricted universe multiverse
# deb https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ jammy-security main restricted universe multiverse
EOF

sudo apt-get update
```

### ARM64 镜像源（用于 RootFS）

在构建时使用 `--apt-mirror` 参数：

```bash
# 阿里云
--apt-mirror https://mirrors.aliyun.com/ubuntu-ports

# 清华
--apt-mirror https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports

# USTC
--apt-mirror https://mirrors.ustc.edu.cn/ubuntu-ports
```

## 🔐 配置 DNS（可选）

如果遇到域名解析问题：

```bash
sudo mkdir -p /etc/systemd/resolved.conf.d
sudo tee /etc/systemd/resolved.conf.d/dns.conf > /dev/null << 'EOF'
[Resolve]
DNS=8.8.8.8 114.114.114.114
FallbackDNS=223.5.5.5 223.6.6.6
EOF

sudo systemctl restart systemd-resolved
```

## 💾 配置交换空间（推荐）

如果内存小于 8GB，建议增加交换空间：

```bash
# 创建 4GB 交换文件
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile

# 永久生效
echo '/swapfile none swap sw 0 0' | sudo tee -a /etc/fstab

# 验证
free -h
```
