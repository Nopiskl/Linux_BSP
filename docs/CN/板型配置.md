# 板型配置说明

板型配置文件位于 `configs/board/` 目录。

## 快速开始

```bash
# 1. 复制模板
cp configs/board/example.conf configs/board/myboard.conf

# 2. 编辑配置
nano configs/board/myboard.conf

# 3. 构建
./build.sh -b myboard
```

## 必需配置

### 基本信息

```bash
BOARD_NAME="myboard"              # 板型名称
ARCH="arm64"                      # arm64 或 arm
KERNEL_GCC="aarch64-linux-gnu-"   # ARM64: aarch64-linux-gnu-
                                  # ARM32: arm-linux-gnueabihf-
```

### 内核配置

```bash
# 内核仓库
LINUX_REPO="https://github.com/torvalds/linux.git"

# 内核版本
LINUX_BRANCH="v6.1"

# 配置文件
LINUX_CONFIG="defconfig"

# 补丁目录（一般填 none）
LINUX_PATHDIR="none"
```

常用内核版本：
- `v6.1` - LTS 长期支持（推荐）
- `v6.6` - 最新 LTS

### Bootloader 配置

#### 方式1：使用已支持的 Bootloader

```bash
# Allwinner SyterKit
BL_CONFIG="sunxi-syterkit"
SYTERKIT_REPO="https://github.com/YuzukiHD/SyterKit.git"
SYTERKIT_BRANCH="master"

# 或 Allwinner U-Boot
BL_CONFIG="sunxi-uboot"
UBOOT_REPO="https://github.com/u-boot/u-boot.git"
UBOOT_BRANCH="master"

# 或 Rockchip U-Boot
BL_CONFIG="rockchip-uboot"
UBOOT_REPO="https://github.com/rockchip-linux/u-boot.git"
UBOOT_BRANCH="master"
```

#### 方式2：自定义 Bootloader

```bash
BL_CONFIG="custom"

# 可选：指定你的 bootloader 仓库
CUSTOM_BL_REPO="https://github.com/your-org/your-bootloader.git"
CUSTOM_BL_BRANCH="main"
```

## 配置示例

### 示例1：Allwinner T527

```bash
BOARD_NAME="t527-dev"
ARCH="arm64"
KERNEL_GCC="aarch64-linux-gnu-"

# 内核
LINUX_REPO="https://github.com/AvaotaSBC/linux.git"
LINUX_BRANCH="linux-5.15"
LINUX_CONFIG="sun55i_t527_bsp_defconfig"
LINUX_PATHDIR="none"

# Bootloader
BL_CONFIG="sunxi-syterkit"
SYTERKIT_REPO="https://github.com/YuzukiHD/SyterKit.git"
SYTERKIT_BRANCH="master"

# 可选
DEVICE_DTS="allwinner/sun55i-t527-devboard"
BOOTARGS="root=LABEL=rootfs rootwait console=ttyS0,115200"
```

### 示例2：官方内核 + 自定义 Bootloader

```bash
BOARD_NAME="myboard"
ARCH="arm64"
KERNEL_GCC="aarch64-linux-gnu-"

# 使用官方内核
LINUX_REPO="https://github.com/torvalds/linux.git"
LINUX_BRANCH="v6.1"
LINUX_CONFIG="defconfig"
LINUX_PATHDIR="none"

# 自定义 Bootloader
BL_CONFIG="custom"
```

```bash
# 设备树文件（不含 .dts 后缀）

# 内核启动参数
BOOTARGS="root=LABEL=rootfs rootwait console=ttyS0,115200"

setup_dhcp(){

```

## 常见问题

### LINUX_CONFIG 填什么？

- 使用内核默认：`defconfig`
- 使用芯片配置：在 `arch/${ARCH}/configs/` 中查找
- 自定义配置：放在 `configs/target/defconfig/` 并指定文件名

### 如何选择内核版本？

推荐使用 LTS 版本：
- v6.1（支持到 2026）
- v6.6（最新 LTS）

### 配置文件出错？

运行清理命令：

```bash
./build.sh clean
```
