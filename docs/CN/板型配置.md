# 板型配置说明

板型配置文件位于 `configs/board/` 目录。

## 快速开始

```bash
# 1. 复制模板
cp configs/board/mainline_soc-example.conf configs/board/myboard.conf

# 2. 编辑配置
nano configs/board/myboard.conf

# 3. 构建
./build.sh -b myboard
```

## 必需配置

### 基本信息

```bash
BOARD_NAME="myboard"              # 板型名称
ARCH="arm64"                      # arm64 或 arm
KERNEL_GCC="aarch64-linux-gnu-"   # ARM64: aarch64-linux-gnu-
                                  # ARM32: arm-linux-gnueabihf-
```

### 内核配置

```bash
# 内核仓库
LINUX_REPO="https://github.com/torvalds/linux.git"

# 内核版本
LINUX_BRANCH="v6.1"

# 配置文件
KERNEL_DEFCONFIG="defconfig"

# 补丁目录（一般填 none）
LINUX_PATHDIR="none"
```

常用内核版本：
- `v6.1` - LTS 长期支持（推荐）
- `v6.6` - 最新 LTS

### Bootloader 配置

#### 使用已支持的 Bootloader

```bash
# 主线 U-Boot
BL_CONFIG="mainline-uboot"
UBOOT_REPO="https://github.com/u-boot/u-boot.git"
UBOOT_BRANCH="master"

# 或 Allwinner U-Boot
BL_CONFIG="sunxi-uboot"
UBOOT_REPO="https://github.com/u-boot/u-boot.git"
UBOOT_BRANCH="master"

# 或 Rockchip U-Boot
BL_CONFIG="rockchip-uboot"
UBOOT_REPO="https://github.com/rockchip-linux/u-boot.git"
UBOOT_BRANCH="master"
```

## 配置示例

### 示例1：Allwinner T527

```bash
BOARD_NAME="t527-dev"
ARCH="arm64"
KERNEL_GCC="aarch64-linux-gnu-"

# 内核
LINUX_REPO="https://github.com/AvaotaSBC/linux.git"
LINUX_BRANCH="linux-5.15"
KERNEL_DEFCONFIG="sun55i_t527_bsp_defconfig"
LINUX_PATHDIR="none"

# Bootloader
BL_CONFIG="mainline-uboot"
UBOOT_REPO="https://github.com/u-boot/u-boot.git"
UBOOT_BRANCH="master"
UBOOT_DEFCONFIG="t527_defconfig"
BUILD_ATF="yes"

# 可选
KERNEL_DTS="allwinner/sun55i-t527-devboard"
UBOOT_DTS="allwinner/sun55i-t527-devboard"
BOOTARGS="root=LABEL=rootfs rootwait console=ttyS0,115200"
```

### 示例2：官方内核 + 主线 U-Boot

```bash
BOARD_NAME="myboard"
ARCH="arm64"
KERNEL_GCC="aarch64-linux-gnu-"

# 使用官方内核
LINUX_REPO="https://github.com/torvalds/linux.git"
LINUX_BRANCH="v6.1"
KERNEL_DEFCONFIG="defconfig"
LINUX_PATHDIR="none"

# Bootloader
BL_CONFIG="mainline-uboot"
UBOOT_REPO="https://github.com/u-boot/u-boot.git"
UBOOT_BRANCH="master"
UBOOT_DEFCONFIG="xxx_defconfig"
ATF_REPO="https://github.com/ARM-software/arm-trusted-firmware.git"
ATF_BRANCH="master"
```

## 常见问题

### KERNEL_DEFCONFIG 填什么？

- 使用内核默认：`defconfig`
- 使用芯片配置：在 `arch/${ARCH}/configs/` 中查找
- 自定义配置：放在 `configs/target/<target-board>/kernel_defconfig`

### 如何选择内核版本？

推荐使用 LTS 版本：
- v6.1（支持到 2026）
- v6.6（最新 LTS）

### 配置文件出错？

运行清理命令：

```bash
./build.sh clean
```
