# 快速开始

本指南帮助你在 5 分钟内开始使用 BSP_T527。

## 系统要求

- Ubuntu 20.04+ 或 Debian 11+
- 4GB+ 内存（推荐 8GB）
- 20GB+ 磁盘空间（推荐 50GB）
- Root 权限（sudo）

## 安装依赖

```bash
sudo apt-get update && sudo apt-get install -y \
    dialog \
    gcc-aarch64-linux-gnu \
    gcc-arm-linux-gnueabihf \
    mmdebstrap \
    debootstrap \
    qemu-user-static \
    binfmt-support \
    build-essential \
    git \
    bc \
    bison \
    flex \
    libssl-dev \
    libncurses-dev \
    kmod
```

## 快速构建

### 方法 1：交互式构建（推荐）

```bash
cd /path/to/BSP_T527
sudo ./build.sh
```

按照提示选择：
1. 板型（Board）
2. 内核目标（Kernel Target）
3. 是否运行 menuconfig
4. 构建模式（完整/仅内核）
5. 是否构建 RootFS
6. 发行版（Ubuntu/Debian）
7. 系统类型（CLI/XFCE/GNOME 等）
8. APT 镜像源

### 方法 2：命令行构建

仅构建内核：
```bash
sudo ./build.sh -b example -o yes
```

构建完整系统（CLI）：
```bash
sudo ./build.sh \
    -b example \
    -r yes \
    -v ubuntu/jammy \
    -t cli \
    -m https://mirrors.aliyun.com/ubuntu-ports
```

构建桌面系统（XFCE）：
```bash
sudo ./build.sh \
    -b example \
    -r yes \
    -v ubuntu/jammy \
    -t xfce \
    -m https://mirrors.tuna.tsinghua.edu.cn/ubuntu-ports
```

## 构建输出

```
output/
├── bootloader-example/         # Bootloader 文件
├── example-kernel-pkgs/        # 内核 .deb 包
├── linux/                      # 内核源码
└── rootfs-*.tar.gz            # RootFS 压缩包（如果构建）
```

## 常见问题

### DNS 解析失败

```bash
sudo mkdir -p /etc/systemd/resolved.conf.d
sudo tee /etc/systemd/resolved.conf.d/dns.conf > /dev/null << 'EOF'
[Resolve]
DNS=8.8.8.8 114.114.114.114
FallbackDNS=223.5.5.5 223.6.6.6
EOF
sudo systemctl restart systemd-resolved
```

### 缺少编译工具链

```bash
sudo apt-get install -y gcc-aarch64-linux-gnu gcc-arm-linux-gnueabihf
```

### 磁盘空间不足

```bash
# 清理构建产物
rm -rf output/

# 清理 APT 缓存
sudo apt-get clean
```
