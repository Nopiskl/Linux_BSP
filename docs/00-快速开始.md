# 🚀 快速开始指南

5 分钟快速上手 BSP_T527 构建系统。

## 📋 前置准备

### 系统要求

- Ubuntu 20.04+ 或 Debian 11+
- 4GB+ 内存（推荐 8GB）
- 20GB+ 磁盘空间
- Root 权限

### 安装依赖

```bash
sudo apt-get update && sudo apt-get install -y \
    dialog \
    gcc-aarch64-linux-gnu \
    mmdebstrap \
    debootstrap \
    qemu-user-static \
    git \
    make \
    bc \
    bison \
    flex \
    libssl-dev \
    libncurses-dev
```

**⏱️ 安装时间：** 约 2-5 分钟

---

## 🎯 方式一：交互式构建（推荐新手）

### 1. 启动构建脚本

```bash
cd /path/to/BSP_T527
sudo ./build.sh
```

### 2. 按照菜单选择配置

```
┌────────── BSP Build Configuration ──────────┐
│                                             │
│ Board:              example                 │
│ Kernel:             linux-6.1               │
│ Build RootFS:       yes                     │
│ Distribution:       ubuntu/jammy            │
│ RootFS Type:        cli                     │
│ APT Mirror:         official                │
│                                             │
│ [  OK  ]                    [  Cancel  ]   │
└─────────────────────────────────────────────┘
```

**推荐配置：**
- 板型：`example`
- 发行版：`ubuntu/jammy` (Ubuntu 22.04)
- 系统类型：`cli` (命令行，最快)
- APT 镜像：`aliyun` 或 `tsinghua` (国内加速)

### 3. 等待构建完成

```bash
✅ Build completed successfully!
Output: /path/to/BSP_T527/output
```

**⏱️ 构建时间：**
- CLI 系统：30-45 分钟
- XFCE 桌面：45-60 分钟

---

## ⚡ 方式二：命令行构建（推荐高级用户）

### 构建最小系统

```bash
sudo ./build.sh \
  --board example \
  --distro-version ubuntu/jammy \
  --rootfs-type cli \
  --apt-mirror https://mirrors.aliyun.com/ubuntu-ports \
  --build-rootfs yes
```

### 仅构建内核

```bash
sudo ./build.sh \
  --board example \
  --kernel linux-6.1 \
  --build-rootfs no
```

### 构建桌面系统

```bash
sudo ./build.sh \
  --board example \
  --distro-version ubuntu/jammy \
  --rootfs-type xfce \
  --build-rootfs yes
```

## 🔄 二次构建（利用缓存）

如果 RootFS 已经构建过，再次运行会自动跳过：

```bash
sudo ./build.sh -b example -r yes -v ubuntu/jammy -t cli

# 输出：
✓ Found existing rootfs tarball, skip build rootfs.
File: output/rootfs-ubuntu-jammy-cli.tar.gz
```

**⚡ 节省时间：** 从 ~30 分钟 → ~5 分钟！

### 强制重新构建

```bash
# 删除缓存的 RootFS
rm output/rootfs-ubuntu-jammy-cli.tar.gz

# 重新构建
sudo ./build.sh -b example -r yes -v ubuntu/jammy -t cli
```

## ❓ 常见问题

### 1. 编译器未找到

```bash
# 错误：compiler 'aarch64-linux-gnu-gcc' not found
# 解决：
sudo apt-get install gcc-aarch64-linux-gnu
```

### 2. 权限错误

```bash
# 错误：Permission denied
# 解决：使用 sudo 运行
sudo ./build.sh
```

### 3. DNS 解析失败

```bash
# 配置 DNS
sudo tee /etc/systemd/resolved.conf.d/dns.conf > /dev/null << 'EOF'
[Resolve]
DNS=8.8.8.8 114.114.114.114
EOF

sudo systemctl restart systemd-resolved
```

### 4. 磁盘空间不足

```bash
# 清理旧构建产物
rm -rf output/

# 清理 Docker（如果使用）
docker system prune -a
```
